# iro -è‰²-

Read hex color code(s) and print

- code itself
- color name (if exists, according to [jonathantneal/color-names](https://github.com/jonathantneal/color-names))
- RGB (both 0-255 and 0-1)
- HSL

## Usage

```
iro [hex color code(s)]
```
